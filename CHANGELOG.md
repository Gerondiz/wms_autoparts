# Changelog

Все значимые изменения в проекте WMS Autoparts будут задокументированы в этом файле.

Формат ведётся в соответствии с [Keep a Changelog](https://keepachangelog.com/ru/1.0.0/),
и проект придерживается [Semantic Versioning](https://semver.org/lang/ru/).

## [1.0.0] - 2026-03-26

### Добавлено

#### Аутентификация и авторизация
- Система аутентификации на базе next-auth v5
- Многоуровневая система ролей (MECHANIC, REPAIR_MANAGER, STOREKEEPER, ADMIN)
- Система разрешений (permissions) для гибкого управления доступом
- Защита маршрутов через middleware

#### Каталог запчастей
- Иерархическая структура категорий запчастей
- Поиск запчастей по названию и артикулу
- Фильтрация по наличию и низким остаткам
- Детальная страница запчасти с изображениями

#### Управление заказами
- Создание заказов на запчасти
- Корзина для добавления нескольких позиций
- Статусы заказов: черновик, на согласовании, одобрен, отклонён, выдан
- Согласование заказов менеджерами
- Выдача запчастей со склада
- Генерация PDF для заказов

#### Складской учёт
- Управление остатками запчастей
- Приход запчастей от поставщиков
- Списание запчастей
- История складских операций
- Отслеживание минимального уровня остатков

#### Админ-панель
- Управление пользователями
- Управление ролями и разрешениями
- Управление иерархией категорий
- Управление запчастями (CRUD)
- Аналитика и отчёты

#### Интернационализация
- Поддержка трёх языков: Русский, English, العربية
- RTL поддержка для арабского языка
- Локализация всех интерфейсов

#### Отчёты и аналитика
- Дашборд с ключевыми метриками
- Отчёты по продажам
- Отчёты по складским операциям
- Аналитика популярности запчастей

### Технологический стек

#### Frontend
- Next.js 14 (App Router)
- React 18.2
- TypeScript 5.x
- MUI (Material-UI) v5
- MUI X DataGrid для таблиц
- MUI X Tree View для иерархий
- Zustand для state management
- SWR для data fetching
- React Hook Form для форм
- Zod для валидации
- @react-pdf/renderer для генерации PDF

#### Backend
- Next.js API Routes
- Drizzle ORM для работы с БД
- PostgreSQL 15+
- next-auth v5 для аутентификации
- bcryptjs для хеширования паролей

#### Инфраструктура
- Docker для контейнеризации
- Docker Compose для оркестрации
- Jest для unit-тестирования
- Playwright для E2E тестирования

### Изменения
- Оптимизирована производительность таблиц с большими данными
- Улучшена обработка ошибок API
- Добавлены loading-состояния для всех страниц

### Исправления
- Исправлены проблемы с типами TypeScript в i18n конфигурации
- Исправлены предупреждения ESLint в hooks
- Исправлены типы в E2E тестах
