# Инструкция для доработок

Приступая к исправлениям, следуй этому процессу:

1. **Выбери задачу**, начиная с критических.
2. **Активируй соответствующего агента**:
   - Для UI/UX: `frontend-developer` или `ui-designer`
   - Для бизнес-логики: `senior-developer`
   - Для тестов: `api-tester` или `test-results-analyzer`
3. **Внеси изменения**, убедись, что исправление затрагивает только указанную проблему.
4. **После правки обязательно проверь**:
   - Запусти тесты, если они есть.
   - Если тестов нет – напиши минимальный тест для проверяемого функционала.
   - Проверь вручную через браузер (если требуется).
5. **Обнови ISSUES.md** – отметь задачу как выполненную (`[x]`).
6. **Переходи к следующей задаче**.

Не пытайся исправлять несколько проблем в одном запросе – это приводит к неполным правкам.

# Замечания по проекту

## UI/UX
- [x] **Левая панель содержит навигационное меню вместо иерархии каталога**
  *Проблема:* В левой части страницы отображается меню вместо дерева категорий.
  *Решение:* HierarchyTree восстановлен в каталоге. Левая панель отображает дерево категорий на десктопе.
  *Приоритет:* критично ✅ ИСПРАВЛЕНО

- [x] **В центральной области не отображаются запчасти для выбранного узла**
  *Проблема:* При клике на узел запчасти не загружаются.
  *Решение:* useParts интегрирован с selectedNodeId. При выборе узла загружаются запчасти.
  *Приоритет:* критично ✅ ИСПРАВЛЕНО

- [x] **Дерево категорий не работает**
  *Проблема:* Ошибка "Maximum update depth exceeded" и "useHierarchy must be used within a HierarchyProvider".
  *Решение:* Удалена зависимость от HierarchyContext. Локальное состояние в HierarchyTree и catalog-content.
  *Приоритет:* критично ✅ ИСПРАВЛЕНО

- [x] **Хлебные крошки отсутствуют или не синхронизированы**
  *Проблема:* Нет хлебных крошек.
  *Решение:* Breadcrumbs отображают путь через useHierarchyPath.
  *Приоритет:* важно ✅ ИСПРАВЛЕНО

- [x] **Header**
  *Проблема:* Header был доступен только на главной странице.
  *Решение:* AppLayout с Header в корневом layout.
  *Приоритет:* критично ✅ ИСПРАВЛЕНО

- [x] **Нужно доработать Header**
  *Проблема:* Выбор языка кнопками, RTL для арабского не работает.
  *Решение:* LanguageSwitcher изменён на dropdown. Добавлена RTL поддержка (flexDirection: 'row-reverse' для ar).
  *Приоритет:* важно ✅ ИСПРАВЛЕНО

- [x] **Мультиязычность**
  *Проблема:* Перевод не работает, отображаются ключи (catalog.title).
  *Решение:* Структура файлов переводов изменена на плоскую. Все ключи на верхнем уровне.
  *Приоритет:* важно ✅ ИСПРАВЛЕНО
